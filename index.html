Definição e utilidade das Bibliotecas

iostream:

Definição: Esta biblioteca em C++ fornece funcionalidades para entrada e saída padrão. Ela inclui objetos como std::cout e std::cin, que são utilizados para projetar na tela

Utilidade: Essa biblioteca é essencial para interagir com o usuário através da linha de comando. Ela permite que você exiba mensagens, solicite entrada do usuário e lide com a entrada e saída de dados em geral.

iomanip:

Definição: A biblioteca iomanip em C++ fornece funcionalidades para manipulação de entrada/saída, para configurar a formatação da saída.

Utilidade: Ela é útil quando precisamos controlar a formatação da saída, como configurar a precisão dos números com casas décimais, definir larguras de campo, alinhar dados. Isso é útil para mostrar dados de forma organizada e legível.

string:

Definição: Esta biblioteca em C++ fornece funcionalidades para manipulação de strings.

Utilidade: Ela é usada para trabalhar com strings de caracteres em C++.

Declaração da estrutura "Item"

  • int numero: Representa o número do produto.
  • string nome: Representa o nome do produto.
  • double precoCompra: Representa o preço de compra do produto.
  • double precoVenda: Representa o preço de venda do produto.
  • int quantidadeEstoque: Representa a quantidade disponível em estoque do produto.
  • int quantidadeComprar: Representa a quantidade que precisa ser comprada do item.
  • int totalVendas: Representa o total de vendas do produto.
  • bool reporEstoque: Representa se o produto precisa ser reposto no estoque, indicado por true ou false.

Definição da Função "imprimirTabela"

Essa função recebe dois parâmetros:

Item itens[]: Um array de itens de papelaria. Cada elemento do array é do tipo Item, que é uma estrutura definida pelo usuário, contendo os detalhes de um item de papelaria.

int tamanho: O tamanho do array de itens, indicando quantos itens estão presentes no array.

Esta função imprime uma tabela bonita com detalhes de cada item de papelaria.

Para cada item na lista:

Calculamos o lucro do item multiplicando a diferença entre o preço de venda e o preço de compra pela quantidade total vendida.

Verificamos se a quantidade em estoque é inferior a 80 e marcamos o campo reporEstoque como verdadeiro ou falso.

Imprimimos os detalhes do item, incluindo o número, nome, preço de compra, preço de venda, quantidade em estoque, quantidade a comprar, total de vendas e se o item precisa ser reposto no estoque.

Função main:

Esta é a função principal do programa.

Criamos uma matriz de itens de papelaria, cada um com detalhes predefinidos, como número, nome, preço de compra, preço de venda, quantidade em estoque, quantidade a comprar e total de vendas.

Chamamos a função imprimirTabela passando a matriz de itens e o tamanho como argumentos.
